Catalyst Library On-Line Help

The Catalyst Library is an Open Source Project. To use the library, you will also need the Coyote Library of David Fanning.

The directories in the Catalyst Library are organized as follows:

  • docs -- This directory contains HTML files documenting the code in the library.
  • resources -- This directory contains Catalyst Library resources (bitmaps)
  • source -- This directory contains the source code for the library
    • applications -- This directory contains Catalyst Library applications
    • coordinates -- This directory contains coordinate objects
    • core -- This directory contains the core objects of the Catalyst Library (e.g., CatAtom)
    • data -- This directory contains data objects (e.g., CatImage)
    • graphics -- This directory contains graphical objects, most for use with interactions
    • interactions -- This directory contains interaction objects
    • ographics -- This directory contains object graphics objects
    • utilities -- This directory contains Catalyst Library utility routines
    • widgets -- This directory contains a complete set of IDL widgets as objects
  • templates -- This directory contains object templates to facilitate further object development
